# Sistema de Gestión de Políticas de Negocio basado en Workflow

Sistema integral para el diseño, ejecución y monitoreo de procesos organizacionales.

## Tecnologías
- **Backend:** Java 17, Spring Boot, Spring Security, MongoDB
- **Frontend:** Angular, TypeScript
- **IA:** Python, FastAPI
- **Infraestructura:** Docker, Docker Compose

## Estructura
- `/backend`: API REST y motor de workflow.
- `/frontend`: Interfaz de usuario y diseñador visual.
- `/ai-service`: Microservicio de asistencia por IA.
- `/database`: Scripts de inicialización de MongoDB.
- `/docs`: Documentación técnica y manuales.
- `/diagrams`: Modelado UML 2.5 en PlantUML.
- `/docker`: Archivos de configuración de contenedores.
